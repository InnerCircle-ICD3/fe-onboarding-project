# 자판기 구현 프로젝트
[프로젝트 요구사항](../../README.md)을 참고하여 바닐라 자바스크립트로 자판기를 구현합니다.

## 기술 스택
- **프론트엔드**
  - Vanilla JavaScript (ES6+)
  - HTML5
  - CSS3

- **개발 도구**
  - Vite (개발 서버)

## 실행 방법
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 작업 상세 내용

- [V] 왼쪽 자판기 금액 표시창
